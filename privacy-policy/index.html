    1. Introduction

    This website is a personal, non-commercial website built using Hugo & Ananke. We respect your privacy and are committed to protecting it.

    This Privacy Policy explains what information is (or is not) collected when you visit this website.

    2. Information We Collect

    We do not actively collect any personal information from visitors.

    Specifically:

    • No user accounts or login systems
    • No contact forms or submission forms
    • No comment systems
    • No analytics tools (Google Analytics, Plausible, Umami, etc.)
    • No AI services or API integrations

    The website is purely for content display.

    3. Log Data

    Like most websites, basic server log data may be automatically collected by the hosting provider (e.g. GitHub Pages), which may include:

    • IP address
    • Browser type
    • Operating system
    • Date and time of access

    This data is controlled by the hosting provider and is not accessed, analyzed, or stored by the website owner.

    4. Cookies

    This website does not use cookies.

    However, the hosting platform or content delivery network (such as GitHub or GitHub Pages) may use essential cookies required for basic website functionality.

    5. Third-Party Services

    This website does not intentionally use any third-party tracking or data collection services.

    Content is hosted via:

    • GitHub Pages

    Please refer to GitHub’s Privacy Policy for details on how they handle data.

    6. Data Security

    Since no personal data is collected, stored, or processed by this website, there is no risk of personal data leakage from the website itself.

    7. Your Rights

    Depending on your jurisdiction, you may have rights under privacy laws such as GDPR.

    As this website does not collect personal data, these rights generally do not apply in practice.

    8. Changes to This Policy

    This Privacy Policy may be updated if the website’s functionality changes. Updates will be reflected on this page.
